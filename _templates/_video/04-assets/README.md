# Assets — Step 7 (human)

Drop generated production files here:

- `images/` — generated stills, named to match the scene number (`01-sceneslug.svg`, `02-…`).
  Export **SVG** from Recraft. Follow `_assets/style-bible.md`; re-attach the Custom Style +
  character sheet on every generation.
- `narration.mp3` — ElevenLabs export using the **locked branded voice** (see style-bible §6).
- `music.mp3` — from **YouTube Audio Library** (or other Content-ID-safe source). **Never** CapCut's
  built-in library. Note the track + attribution in `History/_assets/music/`.

Skim images for style drift (line weight, head shape, accent color) before editing; regenerate outliers.
