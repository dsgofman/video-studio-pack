# Per-Video Template

Copy this whole folder to `<Channel>\<NNN-slug>\` to start a video (e.g.
`History\001-humans-10000-years-ago\`). Work the numbered subfolders in order — they mirror the
13-step pipeline in `CLAUDE.md`:

- `00-packaging/` — Step 2: titles + thumbnail concepts (BEFORE the script)
- `01-research/` — Steps 3–4: research brief + fact-checked sources
- `02-script/` — Step 5: the script
- `03-scenes/` — Step 6: scene list + image prompts
- `04-assets/` — Step 7: generated images, narration, music
- `05-edit/` — Step 8: CapCut edit guide
- `06-metadata/` — Step 9: title, description, chapters, tags
- `07-publish/` — Steps 10–13: Definition of Done + post-publish/analytics

Don't write the script until a title–thumbnail pair survives `00-packaging/`.
